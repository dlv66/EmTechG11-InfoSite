Description

Game streaming (also known as cloud gaming or gaming-on-demand) is a type of gaming aimed at providing end-users of games with smooth and direct playability across different devices. It has much common with video-on-demand services, but interactive. Where one company executes the media on a server, then the output of the media is sent to the screen or to the local device of the user where the user can control the media using a input device, such as a controller or a mouse-and-keyboard combo. The remote server is doing all the hard work while your computer is receiving video (and audio) streaming and sending input orders.
